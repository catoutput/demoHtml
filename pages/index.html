Building a Website with HTML, CSS, and Javascript

Introduction

This page is part of an assignment to convert the demoHtml repo into a GitHub Pages site using self-taught technical skills. The requirements for the site and the methods used to achieve them are outlined below. It was written by catoutput entirely by hand.

Requirement #1: Have a solid color background.

This is accomplished by adding a background-color property to the style of the page's body element. On this page, the background color has a value of #222.

Requirement #2: Have at least one image.

This is accomplished by adding an img element to the page and specifying a path to the desired image with the src attribute. It is also necessary to add an alt attribute to specify text to display in place of the text for readers who cannot view it. On this page, there is an image of a cat from Wikimedia Commons.

Requirement #3: Have at least one hyperlink.

This is accomplished by adding an a elememt to the page and specifying the link's destination with the href attribute. In the interest of accomplishing two tasks at once, click this text for an example of a link to the page on Wikimedia Commons for the cat image used earlier, where you can also read about its license.

Requirement #4: Have at least one snippet of Javascript that adds an OnClick event.

This is accomplished by adding an onclick event to any element, similar to adding an attribute. For example, to trigger an alert box that reads "Hello, World!".

Requirement #5: Use CSS to make your font look pretty!

There are a lot of ways to accomplish this. One way is adding a font-family property to style of the page's body element. On this page, the font family has a value of Tahoma, Verdana, sans-serif, which provdes a few font options depending on what it available on the reader's device.
